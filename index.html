画形状前，打开对其边缘，避免出现虚边；

布尔运算中，减去顶层形状一定要选中上层形状；
分辨率设为72；
弹出渐变对话框时，渐变位置可拖动；
选中若干锚点，进行 cmd + t 变换，可局部变形；
重复复制图层及变换时，隐藏路径与否，决定复制后的图形是否在一个图层；
编组后可转换为矢量智能对象；
定义图案时确认选区没有羽化；
自定义图案可导出为 pat 文件；
当对象无法与辅助线对齐时，关闭首选项中“将矢量工具与变化和像素网格对齐”；
保留暗色，去掉亮色：

保留亮色，去掉暗色：

填充为0时，图层消失，图层样式还在；
图层样式可存储，可导出；
填充纯色，选择色相模式，对下层进行色相修改；
改变色相，可使用剪切蒙版，上层内容，下层范围
将位图转换为智能对象后，大多数滤镜可再次调节；
可利用渐变的模式来画复杂渐变；
矢量蒙版和剪切蒙版可用小白工具继续编辑；
隐藏路径时，复制形状为新建图层；

2017-04-29

PS 快捷键

填充前景色：alt + delet
填充后景色：cmd + delet
缩放工具：z（默认为放大，按住 alt 为缩小）
缩放：alt + 鼠标滚轮
全屏：f
首选项：cmd + k
水平翻转：cmd + y
水平居中：cmd + w
垂直居中：cmd + q
矢量图合层：cmd + e
矢量图分层：cmd + shift + j（需选中形状）
剪切选区到新图层：cmd + shift + j
选择工具：a（按下 cmd 切换小黑小白）
复制图层：cmd + j
复制选区到新图层：cmd + j
从中心：按住 alt
图层合组：cmd + g
钢笔工具下调用小白：按住 cmd
钢笔工具下控制贝兹杆：按住 alt
存为 web 格式：cmd + alt + shift + s
渐变工具：g
裁剪工具：c
复制图层并变形：cmd + alt + t
再次复制图层及变形：cmd + alt + shift + t
拖动圆心：按住 alt
隐藏路径：cmd + shift + h
复制所有可见图层并合并：cmd + alt + shift + e
抠除白色：cmd + alt + 2
打开色阶：cmd + l
反选选区：cmd + shift + i
添加全黑图层蒙版：按住 alt
路径建立选区：cmd + enter
路径建立蒙版：cmd + 蒙版按钮
选区添加至现有蒙版：shift + delete（需选中现有蒙版）
