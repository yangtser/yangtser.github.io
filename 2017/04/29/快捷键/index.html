PS 快捷键

填充前景色：alt + delet
填充后景色：cmd + delet
缩放工具：z（默认为放大，按住 alt 为缩小）
缩放：alt + 鼠标滚轮
全屏：f
首选项：cmd + k
水平翻转：cmd + y
水平居中：cmd + w
垂直居中：cmd + q
矢量图合层：cmd + e
矢量图分层：cmd + shift + j（需选中形状）
剪切选区到新图层：cmd + shift + j
选择工具：a（按下 cmd 切换小黑小白）
复制图层：cmd + j
复制选区到新图层：cmd + j
从中心：按住 alt
图层合组：cmd + g
钢笔工具下调用小白：按住 cmd
钢笔工具下控制贝兹杆：按住 alt
存为 web 格式：cmd + alt + shift + s
渐变工具：g
裁剪工具：c
复制图层并变形：cmd + alt + t
再次复制图层及变形：cmd + alt + shift + t
拖动圆心：按住 alt
隐藏路径：cmd + shift + h
复制所有可见图层并合并：cmd + alt + shift + e
抠除白色：cmd + alt + 2
打开色阶：cmd + l
反选选区：cmd + shift + i
添加全黑图层蒙版：按住 alt
路径建立选区：cmd + enter
路径建立蒙版：cmd + 蒙版按钮
选区添加至现有蒙版：shift + delete（需选中现有蒙版）
