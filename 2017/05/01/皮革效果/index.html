PS 皮革纹理

1.绘制形状，按 d 设置前景黑色背景白色，cmd + delete 将形状填充为白色：

2.打开滤镜库，选择纹理中的彩色玻璃滤镜，调节皮革纹理的大小：

3.将图层透明度调至50%，新建图层并复制相同形状至于下方：

4.对新图层再做一次染色玻璃，得到如下效果：

5.cmd + e 合并两图层，并添加杂色：

6.cmd + a 全选并 cmd + c 复制。打开通道面板新建一个通道。把刚才复制的粘贴上去：

7.点击 RGB 通道返回到图层，点击编辑–填充，在出现的对话框中点击“颜色…”，然后选择需要的颜色：

8.添加光照，点击滤镜–渲染–光照效果，纹理选择 Alpha 1 通道，调节参数：

9.完成。
