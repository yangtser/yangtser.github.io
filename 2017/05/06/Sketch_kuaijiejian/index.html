Sketch 快捷键

发表于 2017-05-06 |

拖拽画布：按住空格
放大区域：z + 左键
缩小区域：z + alt + 左键
复制图层：cmd + d
编组：cmd + g
选择组内图层：cmd + 左键
显示图层距离：按住 alt
新建画板：a
显示／隐藏边框：b
画形状时移动位置：按住空格
缩放全部至视图：cmd + 1
缩放所选至视图：cmd + 2
移动所选至视图中心：cmd + 3
图层上／下移动：cmd + option + up／down
图层移动至顶／底层：cmd + option + control + up／down
屏幕吸色：ctrl + c
演示模式：cmd + .
缩放形状：cmd + k
旋转角度：cmd + shift + r
原位复制：cmd + d
图层重命名：cmd + r
