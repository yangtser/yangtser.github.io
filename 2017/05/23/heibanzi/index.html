PS 黑板字效果

发表于 2017-05-23 |

效果图：

搜索黑板底图

偷个懒，先找一张黑板底图，拖至 PS 中。

制作粉笔字划痕效果

1.新建图层，并填充黑色，转换为智能对象；

2.添加杂色；

3.添加动感模糊，并旋转至合适的角度；

4.cmd + l 调节色阶，增大对比度；

5.将图层模式切换至滤色；

输入文字并建立划痕蒙版

1.输入文字；

2.按住 cmd 点击文字图层建立选区；

3.在划痕图层建立蒙版，并将文字图层填充调低；

4.添加文字描边；

微调，完成！
