PS 毛玻璃效果

发表于 2017-05-21 |

效果图


实现步骤如下：

1.将背景图复制一层，并转换为智能对象；

2.矩形工具画出毛玻璃区域，按住 cmd 点击图层，建立选区；

3.选中背景智能对象图层，点击下方添加蒙版工具，添加矩形可见蒙版；

4.输入文字，同样方法得到文字选区；

5.选中矩形可见蒙版，alt + delete 将文字选区作为不可见区域添加进选区；

6.选中图层，添加高斯模糊；

7.为毛玻璃添加杂色：新建图层，填充为黑色，并转换为智能对象；

8.添加杂色；

9.将杂色图层模式切换为滤色，并可 cmd + l 打开色阶微调毛玻璃亮度；

10.按住 alt 将蒙版复制到杂色图层；

11.完成。
